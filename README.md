# Twine

Directly fetch gifs from [getyarn.io](getyarn.io)

## Installation

```bash
$ cargo install twine-gif
```

## Usage

```
Usage: twine [OPTIONS] <URL>

Arguments:
  <URL>  A valid yarn URL

Options:
  -o, --output <OUTPUT>  A filename to which the gif will be written
  -h, --help             Print help

```
